Listo? perame deja busco una pluma Expliaccion: Como te comente, git es un programa que se conoce como programa controlador de versiones y es una herramienta colaborativa que los programadores la usamos mucho. Hay varios comandos que puedes hacer con git, los mas importantes y que vamos a usar ahorita son: 1. Clone permite descargar un proyecto completo desde tu cuenta o la cuenta de quien sea de github o algun otro servicio que use GIT (Github es el servicio y GIT es el programa) 2. Commit Permite guardar tus cambios de codigo en tu maquina (Es como estar guardando tus avances) 3. Push Una ves que ya tengas todos tus cambios push envia todo el codigo a github 4. Pull Si hay cambios en github este comando los descarga (Por ejemplo si entro yo y hago cambios pues los descargaria) 5. Hay otros comandos como fetch, merge diff, log y otros tantos raritos.
